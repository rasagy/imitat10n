# imitat10n

Collection of exercises created in my Creative Coding module at NID, Bengaluru. 

Part of M.Des. Information Design, I teach a short hands-on, studio style module to introduce design students to creative explorations with code.
I use p5.js, and focus on teaching foundational skills that can help designers learn by playing, recreating, and reimagining.

This is an exhibit for the final exercise, which focuses on recreating and reimagining popular paintings by adding simple interactivity through code.
